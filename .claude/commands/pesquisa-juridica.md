# Pesquisa Jurídica — Legislação, Doutrina e Jurisprudência

Realize pesquisa jurídica aprofundada sobre as questões jurídicas do processo em análise (ou sobre o tema específico solicitado).

$ARGUMENTS

---

## Protocolo de Pesquisa

### ETAPA 1 — Identificação das Questões Jurídicas

Liste todas as questões jurídicas que precisam ser pesquisadas:

```
QUESTÕES JURÍDICAS IDENTIFICADAS
==================================
QJ1: [descrição da questão]
     Relevância: [para qual pedido / aspecto do processo]
     Complexidade: [Simples / Moderada / Alta / Muito Alta]
     
QJ2: [...]
```

### ETAPA 2 — Pesquisa Legislativa

Para CADA questão jurídica:

```
LEGISLAÇÃO APLICÁVEL — QJ[Nº]
===============================

NORMA PRINCIPAL:
- Diploma: [nome da lei]
- Dispositivo: Art. [X], [§/inciso/alínea]
- Texto: "[transcrição do dispositivo]"
- Vigência: [desde quando, alterações relevantes]
- ⚖️ [CONHECIMENTO JURÍDICO EXTERNO]

NORMAS COMPLEMENTARES:
- [mesmo formato acima]

NORMAS CORRELATAS:
- [constitucionais, infraconstitucionais, regulamentares]

CONFLITO DE NORMAS:
- [se houver antinomia, indicar critérios de solução: hierárquico, cronológico, especialidade]

⚠️ ALERTA DE VIGÊNCIA:
- [indicar se alguma norma pode ter sido alterada após minha data de treinamento]
- [se aplicável: "VERIFICAR se houve alteração legislativa recente"]
```

## Regras de Confiabilidade para Pesquisa Jurídica

1. **Legislação**: cite o texto exato do dispositivo quando possível. Indique se pode ter havido alteração recente.
2. **Doutrina**: sempre forneça autor, obra e, idealmente, edição. Se a citação é de memória, indique.
3. **Jurisprudência**: NUNCA invente número de processo. Se não tem certeza do número exato, diga "julgado em sentido similar" e descreva a tese, marcando para verificação.
4. **Diferencie** jurisprudência vinculante (súmula vinculante, ADPF, ADI, RE com repercussão geral, recursos repetitivos) de jurisprudência persuasiva.
5. Quando houver mudança de entendimento (overruling), indique o entendimento anterior e o atual.
