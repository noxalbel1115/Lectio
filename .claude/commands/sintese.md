# Síntese Executiva para Decisão

Produza uma síntese executiva concisa e direta que permita ao magistrado tomar decisão informada sem necessidade de consultar os autos.

$ARGUMENTS

---

## Formato da Síntese

```
╔══════════════════════════════════════════════════════════╗
║              SÍNTESE EXECUTIVA PARA DECISÃO             ║
╠══════════════════════════════════════════════════════════╣
║ Processo: [nº]                                          ║
║ Partes: [autor] x [réu]                                 ║
║ Ação: [tipo]                                            ║
║ Fase: [atual do processo]                               ║
╚══════════════════════════════════════════════════════════╝
```

### 1. EM UMA FRASE
[Resumo da demanda em uma única frase objetiva]

### 2. O QUE O AUTOR QUER
[Lista objetiva dos pedidos, com valores quando aplicável]

### 3. O QUE O RÉU ALEGA
[Defesa central em tópicos]

### 4. PONTO CENTRAL DA CONTROVÉRSIA
[A questão-chave que define o resultado]

### 5. QUADRO PROBATÓRIO RESUMIDO

| Pedido | Prova do Autor | Prova do Réu | Resultado Provável |

- **Tese jurídica aplicável**: [em uma frase]

## Regras para a Síntese

1. **Máximo de 2 páginas** — seja cirúrgico, sem redundâncias
2. Todo fato citado deve ter referência ao processo `[p. XX]`
3. A recomendação deve ser CLARA — evite "depende" sem dar direção
4. Se houver risco significativo de reforma, INDIQUE expressamente
5. Diferencie o que é seguro do que é controverso
6. Use linguagem direta, sem rodeios acadêmicos
